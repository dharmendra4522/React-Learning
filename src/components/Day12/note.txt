‚≠ê INTERVIEW QUESTIONS (Custom Hooks)
Q1. What is a custom hook?

A JavaScript function that uses React hooks inside it and lets you reuse logic.

Q2. Why do we use custom hooks?

To remove duplicate logic and improve code reusability.

Q3. Do custom hooks re-render the component?

Yes, when their internal state changes.

Q4. Why should custom hook name start with ‚Äúuse‚Äù?

React ko pata chale ke ye hook hai ‚Äî warna rules of hooks break honge.

Q5. Can custom hooks return functions?
Yes, they can return:
state
setters
functions
objects
(anything like normal JS functions)
