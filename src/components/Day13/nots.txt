⭐ INTERVIEW QUESTIONS (React Router)
Q1. What is React Router?
A library to handle client-side routing in React apps.

Q2. What is the difference between Link and a tag?
<a> reloads whole page
<Link> prevents reload (SPA)

Q3. What is useParams?
Hook to read dynamic URL values.

Q4. What is useNavigate?
Hook to redirect programmatically.

Q5. What is nested routing?
Routes inside layout wrapper.

Q6. What is 404 route?
Default fallback route (*).
